The Bowler Scala Web Framework v0.1 Released!

January 18th 2011
The first version of Bowler, a RESTful, multi-channel ready web framework in Scala has been released. For details on how to get the code, get started etc please check the homepage.

This is an initial release, which is hopefully usable, but may have some rough edges when it comes to edge case scenarios. Also, some functionality which is planned for a full "1.0" release is still missing, such as scaffolding, UI generation based on a "bean", sbt processors and persistence framework integration extensions for JPA, Squeryl and others.
The purpose of this release is to solicit ideas, feedback and contributions so we may continue improving the framework.
